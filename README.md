# README

# Nyanstagram

Nyanstagram is a social media app where users can upload pictures of cats as well as comment on and react to other users' cats.

# Technologies Used
* HTML
* CSS
* Ruby
* Rails 5.1.6
* Materialize CSS Framework
* Paperclip Gem
* Amazon Web Services
